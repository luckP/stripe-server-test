# Stripe Server

## Setup

1. Go to project folder

> cd server

2. Install Dependencies

> npm run setup

3. Create .env with following content

````
DB_URL=

PORT=3000
SECRET=AAA
STRIPE_SECRET_KEY=sk_test...

````

## Run

1. Start Server

> npm run dev

2. Open in url or using an Endpoint test (postman)

> http://127.0.0.1:3000/
